# Netflix Exploratory Data Analysis (EDA)

## Objective
Analyze Netflix dataset to identify trends, patterns, and insights using Exploratory Data Analysis techniques.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Tasks Performed
- Data Cleaning
- Missing Value Handling
- Statistical Summaries
- Trend Analysis
- Visualization
- Correlation Analysis

## Visualizations Included
- Movies vs TV Shows
- Top Countries
- Ratings Distribution
- Content Added Over Years
- Top Genres
- Top Directors
- Release Trends

## Key Insights
- Movies dominate Netflix content.
- US contributes the most titles.
- Netflix content growth accelerated after 2015.
- Drama-related genres are highly common.

## Conclusion
The project demonstrates analytical thinking and exploratory data analysis skills using real-world streaming data.
